**KFuji RGB-DS** is a dataset for object detection tasks. It is used in the agricultural industry. 

The dataset consists of 967 images with 13835 labeled objects belonging to 1 single class (*apple*).

Images in the KFuji RGB-DS dataset have bounding box annotations. All images are labeled (i.e. with annotations). There are 3 splits in the dataset: *val* (155 images), *train* (619 images), and *test* (193 images).
